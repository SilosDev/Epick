## Getting Started

### Installation

1. Clone the repository:
```bash
git clone https://github.com/SilosDev/Epick.git
cd Epick
```

2. Build the project:
```bash
dotnet build
```

3. Run the application:
```bash
dotnet run
```

### Running Tests

```bash
dotnet test
```
